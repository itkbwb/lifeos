data/lifeos.db

## Обновление с v0.1.0 на Raspberry Pi

Сначала останови старый сервер сочетанием Ctrl+C.

Сделай резервную копию старой базы:

```bash
cd ~/lifeos_mvp
cp data/lifeos.db ~/lifeos_backup_before_v0.2.0.db
```

Распакуй новую версию в новую папку:

```bash
cd ~
unzip lifeos_v0.2.0.zip
```

При необходимости перенеси старую базу:

```bash
cp ~/lifeos_mvp/data/lifeos.db ~/lifeos_v0.2.0/data/lifeos.db
```

Запусти:

```bash
cd ~/lifeos_v0.2.0
chmod +x run_linux.sh install_service.sh backup_db.sh
./run_linux.sh
```

Открой:

```text
http://IP_МАЛИНЫ:8000
```

Для твоей текущей сети адрес выглядит примерно так:

```text
http://192.168.219.107:8000
```

## Постоянный запуск после перезагрузки

Останови ручной сервер через Ctrl+C, затем:

```bash
cd ~/lifeos_v0.2.0
./install_service.sh
```

Проверка:

```bash
sudo systemctl status lifeos
```

Логи:

```bash
journalctl -u lifeos -f
```

Перезапуск:

```bash
sudo systemctl restart lifeos
```

## PWA

Для полноценной установки PWA нужен HTTPS или localhost.

По обычному локальному HTTP интерфейс работает, но Chrome может не показать установку.

Постоянный HTTPS адрес и авторизация будут следующим этапом.
